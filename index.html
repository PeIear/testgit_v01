Обо мне

Я выпускник специалитета «РТУ МИРЭА», кафедра "Радиотехнических и телекоммуникационных систем" (новое название кафедры "Институт радиоэлектроники и информатики").

Проходил курсы по аналитике SQL по реляционным БД, делаю запросы к БД на языке MicrosoftSQL, пользуюсь методами подзапросов и сложных join'ов. Имею базовые знания по сетям и администрированию.

Умею пользоваться виртуальными машинами - через VirtualBox запускаю гостевые ОС Ubuntu Linux на ПК. Также настраиваю удаленное подключени к гостевой системе через протокол ssh (по паролю или по заранее сгенерированным ключам).

Имеются базовые знания bash. Умею свободно пользоваться терминалом - а именно пользоваться базовыми командами, взаимодействовать и управлять файлами и директориями, а также управлять правами доступа. Также умею грепать, пользуюсь текстовым редактором VIM (пока что только в качестве редактора, могу также использовать nano). Умею пользоваться пайплайнами. Знаком с переменными окружения и volumes (больше по практике с докером). Умею пользоваться базовым пакетным менеджером, устанавливать программы с основого репозитория (допустим, nginx, apache).

Также умею устанавливать и работать с docker ( также с docker compose, а именно работа c файлами формата yml) - умею запускать контейнеры по готовым образам (такие как nginx, apache tomcat, редактор flame, heimdal, sql и др.) с официального репозитория (dockerhub). Помимо остального также в докере умею создавать свои dockerfile на основе ubuntu, php, python и т.п., а также на основе dockerfile и, если требуется, файлов формата sh, php, py - могу собирать образы, docker image. Помимо всего умею создавать отдельные сети типа bridge.

Также к вышеупомянотому умею запускать стеки контейнеров через docker compose, используя как готовые файлы формата yml с dockerhub, так и могу самостоятельно создавать yml файлы. В дополнение через docker compose запускаю Portainer, через которого также умею запускать контейнеры или стеки, а также строить образы.

Также могу настраивать конфиги nginx для запуска веб сервера со своей страницой, допустим, на основе html, css и js. Самостоятельно изучаю и практикуюсь в программировании на языке JavaScript.

Имею способности к быстрому обучению и высокой работоспособности, а также желание постоянно развиваться. С легкостью выполняю поставленные задачи и цели, всегда стремлюсь выполнять начатое дело до конца. Ответственный, пунктуальный, с легкостью нахожу общий язык в любом коллективе.
